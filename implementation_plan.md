- **核心定位**：现代投资组合理论 —— 有效前沿的边界
- **核心公式**：投资组合方差 $\sigma_p^2 = \sum \sum w_i w_j \sigma_i \sigma_j \rho_{ij}$
- **痛点切入**：“我买了10只股票分散风险，为什么大跌时还是一起腰斩？”
- **起源故事**：1952年，25岁的马科维茨在芝加哥大学图书馆读到《投资价值理论》时，灵光乍现：风险不是单个资产的波动，而是它们如何共同波动。
- **四大类比**：
  1. 拔河比赛（协方差/相关性）：全找大力士不如找不同方向用力的人
  2. 冰淇淋与雨伞摊（对冲风险）：天晴卖冰淇淋，下雨卖雨伞，收益稳定
  3. 调色板混色（非线性收益）：红+蓝=紫，组合风险小于单资产加权
  4. 餐厅套餐搭配（有效前沿）：营养与价格的最优边界
- **著名使用者**：瑞·达利欧（Ray Dalio - 全天候策略）、大卫·斯文森（David Swensen - 耶鲁模式）
- **六大场景**：股债60/40配置、A股与美股QDII对冲、加密货币与黄金配置等，并**配有一张实战场景应用图**。
