#### [NEW] `episodes/第12篇_夏普比率_策略质量的标准尺.md`
将遵循《写作规范指南》的黄金结构（去除最后的导航节），核心变量如下：

- **核心定位**：夏普比率 —— 策略质量的标准尺
- **核心公式**：$SR = \frac{R_p - R_f}{\sigma_p}$
- **痛点切入**：朋友向你吹嘘他今年赚了50%，你该羡慕吗？（如果你知道他用了5倍杠杆全仓单只股票，承受了90%的回撤风险，你不仅不会羡慕，还会觉得他是个赌徒）。
- **起源故事**：1966年，威廉·夏普如何给整个华尔街的基金经理制定了“KPI”。
- **四大类比**：
  1. 奥运体操评分（难度系数与完成度）：收益是动作，风险是难度系数。
  2. 汽车油耗表（性价比）：百公里加速快（高收益）但也可能百公里100个油（高风险）。
  3. 拳击手抗击打测试。
  4. 贷款买房的杠杆还原。
- **著名使用者**：文艺复兴科技（大奖章基金超高夏普比率的神话）、各大顶尖对冲基金。
- **实战场景**：如何用夏普比率筛选公募基金、如何判断一个量化策略是否过拟合、多策略对比等，**配有一张实战场景应用图**。
